# Project Summary
Green Community 225 is a full-stack web application aimed at promoting environmental protection and biodiversity in Côte d'Ivoire. It serves as a platform for an NGO that mobilizes youth through various community-driven projects, including tree planting, educational workshops, and donations to support their initiatives.

# Project Module Description
The project consists of a backend service for managing data and a frontend interface for user interaction. Key functionalities include:
- User authentication and management
- Project listings and management
- Donation processing
- Testimonials and gallery management
- Contact and feedback systems

# Directory Tree
```
app/
├── backend/
│   ├── core/                # Core functionalities (auth, database, config)
│   ├── data_models/         # JSON data models for various entities
│   ├── dependencies/        # Dependency management
│   ├── main.py              # Main entry point for the backend application
│   ├── routers/             # API route definitions
│   ├── services/            # Business logic and data handling
│   ├── requirements.txt      # Backend dependencies
│   └── alembic/             # Database migrations
└── frontend/
    ├── src/                 # Source code for the frontend application
    ├── public/              # Public assets (images, favicon)
    ├── package.json          # Frontend dependencies and scripts
    └── vite.config.ts        # Vite configuration for building the frontend
```

# File Description Inventory
- **backend/**
  - `main.py`: Entry point for the backend application.
  - `alembic/`: Directory containing migration scripts for database schema changes.
  - `routers/`: Contains API route definitions for handling requests.
  - `services/`: Contains business logic for handling data operations.
  
- **frontend/**
  - `src/`: Contains React components, pages, and application logic.
  - `public/`: Contains static assets like images and favicon.
  - `package.json`: Lists dependencies and scripts for building and running the frontend.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - Navigate to the backend directory and run:
     ```bash
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to the frontend directory and run:
     ```bash
     pnpm install
     ```

2. **Run the Application**:
   - Start the backend server:
     ```bash
     python main.py
     ```
   - Start the frontend development server:
     ```bash
     pnpm run dev
     ```

3. **Build the Application**:
   - For production builds, run the following in the frontend directory:
     ```bash
     pnpm run build
     ```
